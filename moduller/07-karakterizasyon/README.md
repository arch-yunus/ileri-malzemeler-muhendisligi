# 🔍 Modül 07: Malzeme Karakterizasyon Teknikleri

Gözle görülemeyen dünyayı (mikro ve nano ölçek) anlamak için kullanılan ileri analitik yöntemler.

## 🎯 Ders Hedefleri
- Mikroskobik görüntüleme tekniklerini karşılaştırmak.
- Kristal yapı ve kimyasal kompozisyon analizini öğrenmek.

## 📖 Konu Başlıkları

### 1. Elektron Mikroskopisi
- **SEM (Scanning Electron Microscopy):** Yüzey morfolojisi.
- **TEM (Transmission Electron Microscopy):** Atomik çözünürlükte iç yapı.

### 2. X-Işını Kırınımı (XRD)
- Faz analizi ve kristalografik yapı tayini.
- Bragg Yasası uygulamaları.

### 3. Spektroskopik Teknikler
- **EDS/EDX:** Kimyasal element analizi.
- **FTIR:** Polimerik bağ analizi.

### 4. Yüzey Analizi (AFM/STM)
- Atomik kuvvet mikroskobu ile yüzey pürüzlülüğü.
